jQuery event.data Property In jQuery, we have learned that the event handler function automatically receives the event object. We have discussed a number of built-in properties on the event object. In this article, we will see the event.data property of the event object. This property helps to pass any data to the event handler function. Syntax: We have learned the syntax of the event handler earlier using the on() method. $(selector).on(event, childSelector, data, function, map) This data parameter is the data in the form of an object which we want to pass to the event handler function. When we will want to use that data, we can retrieve that by event.data.
